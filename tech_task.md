Макет сайта Wolfor Koadon (структура)
Главная страница (Landing)
Блок Содержание
Шапка (хедер) Логотип (слева). Навигация: Книги / Игры / Лор / Новости / О проекте (справа). Крупная кнопка «Войти в мир» (CTA).
Герой-блок Полноэкранный фон (арт вселенной). Крупный заголовок: «Добро пожаловать в мир Wolfor Koadon». Подзаголовок: «Книги. Игры. Вселенная. Одна история». Кнопка «Исследовать» (скролл вниз).
3 крупные карточки (с иконками) Книги → «Погрузитесь в цикл «Сэт». Игры → «Скоро. Игра по вселенной». Лор → «История, кланы, артефакты».
Цитата / слоган Крупный текст: «Справедливость не умирает. Она ждёт своего часа.» (или другая ключевая фраза из вселенной).
Последние новости 2-3 карточки новостей (анонсы, выход книг, dev-логи по игре).
Подвал (футер) Логотип. Ссылки: Книги, Игры, Лор, Новости, Контакты. Иконки соцсетей (VK, Telegram, YouTube, Discord). Копирайт.
Раздел «Книги»
Стратегия: Показать все книги серии «Сэт» в виде крупных карточек. Быстрый доступ к описанию, покупке, фрагментам лора.

Страница /books
Элемент Описание
Заголовок «Книги цикла «Сэт»
Фильтры (опционально) По хронологии / по дате выхода / по кланам (если применимо)
Карточки книг Крупная обложка. Название. Короткое описание (одна строка). Кнопка «Подробнее».
При клике на «Подробнее» Страница книги с полным описанием, отрывком, ссылкой на покупку (бумажная / электронная), цитатами, связью с лором (какие кланы, события).
Раздел «Игры»
Стратегия: Показать текущий проект (игра «Лариар») и будущие планы. Dev-логи, скриншоты, механики.

Страница /games
Элемент Описание
Заголовок «Игры по вселенной Wolfor Koadon»
Крупный баннер Проект «Лариар» (Darkest Dungeon-подобная игра). Логотип, жанр, статус (в разработке).
Блок «Об игре» Короткое описание: сюжет, жанр, ключевые механики.
Блок «Геймплей» Скриншоты (карусель) + ссылка на Dev-логи (посты VK / Telegram).
Блок «Кланы и герои» Краткий обзор 6 кланов (Экионар, Нуан, Треот, Траэн, Фадэир, Дуанар). Иконки + короткое описание.
Блок «Новости разработки» Список последних Dev-логов со ссылками на соцсети.
Блок «Поддержать проект» Кнопка «Следить за разработкой» (ссылка на канал / группу).
Раздел «Лор»
Стратегия: Энциклопедия вселенной. Удобная навигация по разделам. Возможность читать как энциклопедию.

Страница /lore
Элемент Описание
Заголовок «Лор вселенной Wolfor Koadon»
Крупные разделы (сетка 2-3 колонки) Карточки: «Сэты» (кто они, кланы, Вертуджья), «Акиара» (мир, планета, энергопотоки), «Униер» (враги, паразиты, биопаразиты), «Империя Око» (сактаны, Аркрад, Иергом), «Персонажи» (главные герои книг и игры), «Артефакты» (маски, камни, Тарнары).
При клике на раздел Открывается страница с текстом, иллюстрациями, ссылками на связанные разделы.
Поиск Крупное поле поиска по лору (поиск по ключевым словам: «Шакриз», «Вольфор», «Вакаай» и т.д.).
Визуальная карта мира Карта с кликабельными областями (локации книг и игры).
Раздел «Новости»
Стратегия: Лента новостей проекта. Всё в одном месте: анонсы книг, Dev-логи, конкурсы, события.

Рекомендации по логистике и юзабилити
Принцип Как реализовано
Крупные окна Блоки занимают всю ширину экрана. Картинки большие. Текст минимален на главной, подробности — глубже.
Минимализм Нет лишних декоративных элементов. Только контент, воздух, крупные кнопки.
Удобство Меню всегда видно (фиксированная шапка). Поиск — в шапке (крупное поле). Хлебные крошки на внутренних страницах.
Доступность к контенту Контент структурирован по трём основным разделам (Книги → Игры → Лор). Всё, что нужно, — максимум за 2 клика.
Мобильная адаптация Меню сворачивается в бургер. Блоки перестраиваются вертикально.
Скорость загрузки Картинки в webp. Тяжёлые элементы — ленивая загрузка.
